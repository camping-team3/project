# domain/board/

## 목적
캠핑장 운영에 필요한 커뮤니티 및 콘텐츠 관리 (공지사항, 리뷰, Q&A).

## 주요 파일
| 파일명 | 설명 |
|--------|------|
| `Board.java` | 게시판 엔티티 (공용 또는 상속 구조 검토 중) |

## AI 작업 지침
- **리뷰 권한**: 실제 이용 고객(`CHECKED_OUT`)만 리뷰를 작성할 수 있도록 비즈니스 로직에서 검증한다.
- **Q&A**: 사용자가 질문을 등록하고 관리자가 답변을 달면 `is_answered` 상태를 변경한다.
- **이미지**: 리뷰나 갤러리 게시물은 `Image` 엔티티와 1:N 관계를 맺는다.

## 의존성
- 내부: `domain/user` (작성자), `domain/reservation` (리뷰 대상)
