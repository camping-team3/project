# 리뷰 목록 페이지 기본 이미지 추가 리포트
Date: 2026-03-25
Reporter: G
Status: 완료

## 1. 작업 요약
리뷰 목록 페이지(`review/list.mustache`)에서 사진이 없는 리뷰 카드에 기본 이미지를 적용하여 시각적 완성도를 높이고 전체적인 레이아웃의 통일성을 확보하였습니다.

## 2. 변경 사항
### UI/UX
- `src/main/resources/templates/review/list.mustache`
  - `{{^images}}` 블록 내부에 기본 이미지(`/images/camping_review2.jpg`)를 추가.
  - 기본 이미지 위에 `image` 아이콘을 투명하게 겹쳐 사진이 없는 리뷰임을 직관적으로 표현.
  - `ratio-16x9` 및 `object-fit-cover` 클래스를 유지하여 사진이 있는 리뷰와 동일한 카드 비율을 유지.

## 3. 검증 결과
- 사진이 없는 리뷰 카드에서 회색 배경 대신 캠핑장 배경 이미지가 정상적으로 노출됨을 코드상으로 확인.
- 모든 리뷰 카드가 동일한 높이와 비율을 가지게 되어 목록 화면이 훨씬 정돈됨.

## 4. 비유로 설명하는 이번 작업
사진이 없는 리뷰 카드는 마치 '사진 준비 중'이라는 푯말만 덜렁 놓여 있는 빈 액자 같았습니다. 이번 작업을 통해 그 액자 안에 예쁜 풍경 그림(기본 이미지)을 끼워 넣어, 전체 전시관(리뷰 목록)의 분위기를 한결 화사하게 만든 것과 같습니다.
