# Report: 관리자 미구현 메뉴 정리

**Date:** 2026-03-25
**Reporter:** KimNaKim
**Status:** Completed

## 1. 작업 개요
관리자 페이지의 사이드바 메뉴 중, 현재 요금 계산 로직이 미구현된 '시즌 관리' 항목을 삭제하여 사용자(관리자)의 혼선을 방지하고 시스템의 완성도를 높였습니다.

## 2. 변경 사항
- **파일 경로:** `src/main/resources/templates/layout/admin-header.mustache`
- **수정 내용:** 사이드바 내의 `시즌 관리` 링크(`<a>` 태그) 블록을 완전히 제거하였습니다.

## 3. 검증 결과
- 관리자 페이지 접속 시 사이드바에서 '시즌 관리' 메뉴가 더 이상 노출되지 않음을 확인하였습니다.
- 메뉴 삭제 후에도 사이드바의 다른 항목(구역/사이트 관리, 공지사항 관리 등)의 레이아웃이 깨지지 않고 정상적으로 유지됨을 확인하였습니다.

## 4. 비유로 설명하는 작업 내용
마치 건물의 안내판에 아직 공사 중이라 들어갈 수 없는 방의 이름이 적혀 있어 손님들이 헷갈려 하던 상황이었습니다. 이번 작업은 안내판에서 해당 방의 이름을 지워 손님들이 헛걸음하지 않도록 배려한 것과 같습니다. 나중에 방이 완공되면(기능이 구현되면) 그때 다시 안내판에 이름을 적어 넣으면 됩니다.
