# 📝 리포트: 관리자 대시보드 미답변 QnA 현황 실시간 연동

**날짜**: 2026-03-20
**Reporter**: KimNaKim
**작업 단계**: Phase 3 - 관리자 기능 고도화

## 1. 작업 요약
- 관리자 홈 대시보드에 정적으로 표시되던 미답변 QnA 개수와 목록을 실제 데이터베이스 데이터와 실시간 연동하였습니다.

## 2. 변경 사항

### 2.1 백엔드 로직 (`AdminController`)
- **데이터 바인딩**: 대시보드(`@GetMapping("/admin")`) 진입 시 `QnaService`를 호출하도록 수정하였습니다.
- **통계 연동**: 전체 미답변 건수(`unansweredCount`)를 모델에 추가하였습니다.
- **목록 연동**: 최신 미답변 게시글 5건(`unansweredQnas`)을 페이징 로직을 활용해 조회하여 모델에 추가하였습니다.

### 2.2 프론트엔드 UI (`dashboard.mustache`)
- **요약 카드 동적화**: 하드코딩되어 있던 숫자를 `{{unansweredCount}}` 변수로 교체하였습니다.
- **실시간 목록 렌더링**: `{{#unansweredQnas}}` 루프를 적용하여 작성자 이름, 작성일, 제목이 동적으로 출력되도록 개선하였습니다.
- **내비게이션 연결**: 목록 아이템 클릭 시 또는 '답변 달러 가기' 버튼 클릭 시 해당 QnA의 상세 답변 페이지로 즉시 이동하도록 구현하였습니다.
- **예외 처리**: 미답변 데이터가 없을 경우를 대비한 안내 문구 섹션을 추가하였습니다.

## 3. 검증 결과
- **정확성**: 관리자 홈 접속 시 상단 요약 카드의 숫자가 현재 DB 내 미답변 문의 개수와 실시간으로 일치함을 확인하였습니다.
- **정렬**: 미답변 목록이 최신순으로 상단에 노출되며, 최대 5개까지만 표시되는 것을 확인하였습니다.
- **사용성**: 목록 클릭 시 상세 페이지로의 리다이렉션이 정상 동작함을 확인하였습니다.

## 4. 어려운 내용 설명 (비유로 설명하기)
### 📢 실시간 업무 상황판 설치
이전의 대시보드가 "아침에 한 번 확인하고 적어둔 메모지"였다면, 이제는 **"사무실 정면에 설치된 실시간 디지털 전광판"**과 같습니다. 손님이 질문을 올리는 즉시 전광판의 숫자가 올라가고, 가장 급하게 처리해야 할 할 일들이 전광판 상단에 실시간으로 떠오릅니다. 덕분에 관리자는 일일이 목록을 뒤져보지 않아도 현재 어떤 업무가 밀려 있는지 한눈에 파악하고 즉각 대응할 수 있게 되었습니다!
