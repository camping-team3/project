# Report: 어드민 QnA 관리 페이지 UI 간소화 (내보내기 버튼 제거)
Date: 2026-03-23
Reporter: Gemini CLI

## 1. 작업 요약
어드민 QnA 관리 페이지에서 구현되지 않은 채 방치되어 있던 '내보내기 (Excel)' 버튼을 제거하여 UI의 완성도를 높이고 불필요한 요소를 정리했습니다.

## 2. 변경 사항
- **템플릿 수정**:
  - `src/main/resources/templates/admin/qna/list.mustache` 파일 수정.
  - 상단 헤더 영역의 `내보내기 (Excel)` 버튼(`button` 태그) 삭제.
  - 버튼 삭제 후 제목(`질의응답 관리`)과 설명 영역이 좌측에 깔끔하게 정렬되도록 레이아웃 유지.

## 3. 비유를 통한 설명
진열대 위에 '수리 중'이라는 푯말이 붙은 채 방치되어 있던 오래된 기계(미구현 버튼)를 치운 것과 같습니다. 덕분에 매장(페이지)이 더 정돈되어 보이고 손님(관리자)들이 헷갈리지 않게 되었습니다.

## 4. 검증 결과
- QnA 관리 페이지 상단에서 더 이상 엑셀 내보내기 버튼이 노출되지 않음을 확인했습니다.
- 버튼이 사라진 후에도 나머지 탭 메뉴 및 테이블 레이아웃이 정상적으로 작동함을 확인했습니다.
