# AI-Based Review Management System Final Report

**Date:** 2026-03-23
**Reporter:** handikim
**Phase:** Phase 4 - AI 기반 리뷰 관리 및 평점 시스템 고도화 완료

---

## 📝 프로젝트 요약
사용자의 리뷰 등록 시 AI(Gemini)와 서버 필터를 연동하여 비방 위험도를 분석하고, 관리자가 효율적으로 리뷰를 관리하며 평점을 유지할 수 있는 지능형 리뷰 관리 시스템을 성공적으로 구축했습니다.

## 🛠️ 주요 구현 내용

### 1. 지능형 비방 탐지 엔진
- **2단계 필터링**: 서버 측 비속어 블랙리스트(ProfanityFilter)로 즉시 차단하고, 문맥적 비방은 Gemini API(AiAnalysisService)로 1~5점 점수를 매깁니다.
- **비동기 처리**: 리뷰 등록 속도 저하를 막기 위해 `@Async`를 사용하여 백그라운드에서 분석을 수행합니다.

### 2. 관리자 지휘 본부 (Dashboard)
- **위험군 우선 노출**: 위험도 3점 이상 리뷰를 빨간색으로 강조하여 관리자가 즉시 조치할 수 있도록 UX를 설계했습니다.
- **원클릭 조치**: [유지] 또는 [삭제(사유 입력)] 버튼을 통해 신속한 관리가 가능하며, 모든 조치는 논리 삭제(Soft Delete) 방식으로 기록을 보존합니다.

### 3. 실시간 평점 동기화
- **정확한 재계산**: 관리자의 삭제/복구 액션 시 해당 캠핑장의 평점과 리뷰 수를 즉시 재계산하여 데이터 무결성을 보장합니다.

### 4. 투명한 사용자 피드백
- **마이페이지 알림**: 삭제된 리뷰를 본인에게 노출하며 관리자의 삭제 사유를 함께 보여줌으로써 운영의 투명성을 확보했습니다.

## ✅ 최종 검증 결과
- 비속어 포함 리뷰 등록 시 관리자 페이지에 즉시 '5점' 및 강조 표시됨 확인.
- 관리자 삭제 처리 후 공용 목록에서 사라지고 캠핑장 평점이 정확히 차감됨 확인.
- 마이페이지에서 본인의 삭제된 리뷰와 사유가 시각적으로 잘 구분되어 표시됨 확인.

## 💡 어려운 내용 설명 (비유)
이번 시스템은 **'지능형 CCTV가 달린 아파트 게시판'**과 같습니다.
- 주민(사용자)이 글을 쓰면, CCTV(AI)가 즉시 글의 내용을 읽어봅니다.
- 욕설이나 비방이 보이면 경비실(관리자 페이지)에 즉시 빨간 불이 들어옵니다.
- 경비원(관리자)은 빨간 불이 들어온 글을 확인하고, "이건 지워야겠군" 하고 쪽지(삭제 사유)를 붙여서 창고(Soft Delete)로 옮깁니다.
- 게시판 앞(공용 목록)에서는 글이 사라졌지만, 글을 쓴 주민은 창고에 가서 왜 내 글이 옮겨졌는지 경비원의 쪽지를 확인할 수 있습니다.
이 모든 과정이 자동으로 이루어져 게시판은 항상 깨끗하게 유지됩니다.
