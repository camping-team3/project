# Report: 어드민 헤더 통합 검색 제거 및 홈 버튼 추가
Date: 2026-03-23
Reporter: Gemini CLI

## 1. 작업 요약
어드민 페이지 헤더에서 불필요한 기능(검색, 알림, 설정)을 제거하고, 대신 관리자가 메인 페이지로 쉽게 나갈 수 있도록 '홈으로 돌아가기' 버튼을 추가하여 편의성을 개선했습니다.

## 2. 변경 사항
- **템플릿 수정**:
  - `src/main/resources/templates/layout/admin-header.mustache` 파일 수정.
  - 불필요한 `search-wrapper`, 알림 및 설정 아이콘 버튼 삭제.
  - 좌측 상단에 홈 아이콘(`home`)이 포함된 '홈으로 돌아가기' 버튼(`<a>` 태그) 추가.
  - `.btn-outline-fh` 스타일을 적용하여 어드민 대시보드와 조화로운 디자인 유지.

## 3. 비유를 통한 설명
책상 위에서 잘 쓰지 않는 큰 연필꽂이와 자질구레한 소품들을 치우고, 그 자리에 거실로 바로 나갈 수 있는 편리한 문(홈 버튼)을 만든 것과 같습니다. 방 안에서 업무를 보다가 언제든 밖으로 나갈 수 있게 되었습니다.

## 4. 검증 결과
- 어드민 헤더 왼쪽에 '홈으로 돌아가기' 버튼이 정상적으로 노출됨을 확인했습니다.
- 버튼 클릭 시 메인 페이지(`/`)로 정확히 이동함을 확인했습니다.
