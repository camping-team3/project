# Report: 예약 변경 페이지 UI 정밀 조정

- **Date**: 2026-03-24
- **Reporter**: parkcoding
- **Task**: 마이페이지 예약 변경 신청 폼 레이아웃 및 정보 표시 최적화

## 📝 작업 요약
예약 변경 신청 페이지(`reservation-change.mustache`)의 사용성 개선을 위해 기존 예약 정보 가시성을 높이고, 폼 레이아웃을 효율적으로 재배치했습니다.

## 🛠️ 변경 사항
1. **기존 예약 정보**:
   - 상단 뱃지의 텍스트를 '예약 완료'에서 실제 예약 번호(`RESERV-{{id}}`)로 변경하여 정보를 구체화했습니다.
2. **폼 레이아웃 최적화**:
   - `form` 태그에 `p-4` 패딩을 추가하여 입력 필드들이 테두리에 밀착되지 않도록 여백을 확보했습니다.
   - 체크인 날짜, 체크아웃 날짜, 인원 선택 필드를 기존의 2줄(2:1) 구조에서 1줄(3분할, `col-md-4`) 구조로 통합하여 수직 공간을 절약하고 가독성을 높였습니다.

## ✅ 검증 결과
- 예약 번호가 뱃지 내에 정상적으로 출력됨을 확인했습니다.
- 입력 폼 내부의 여백이 확보되어 시각적으로 안정감이 생겼습니다.
- 주요 입력 필드 3종이 한 줄에 균형 있게 배치되어 폼 전체의 길이가 짧아지고 사용자 경험이 개선되었습니다.
