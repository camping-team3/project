# Review Compound Filtering System Implementation Report

**작성자:** 김현지
**날짜:** 2026-03-24

## 작업 요약
- 명백한 욕설은 즉시 차단하고, 문맥적 비방은 AI가 사후 분석하는 **'컴파운드 필터링 모드'**를 성공적으로 구현하였습니다.

## 변경 사항
- **ProfanityFilter.java (L1 강화)**:
    - 정규표현식(`[\\s!@#$%^&*()\\-_=+\\[\\]{};':\",.<>/?0-9]`)을 사용하여 공백, 특수문자, 숫자를 제거한 클린 텍스트로 비속어를 검사하도록 로직을 강화하였습니다.
    - 이를 통해 "시!발", "개 2새끼"와 같은 우회 시도를 효과적으로 방어합니다.
- **ReviewService.java (컴파운드 로직 적용)**:
    - `save` (등록) 및 `update` (수정) 메서드 최상단에 실시간 비속어 필터링 로직을 추가하여 위반 시 `Exception400`을 던지도록 하였습니다.
    - 수정(`update`) 완료 후에도 `aiAnalysisService.analyzeReviewAsync`를 호출하여 문맥 분석을 수행하도록 개선하였습니다.
- **ReviewController.java**:
    - 예외 타입을 `Exception400`으로 통일하여 사용자에게 로그인 리다이렉트 대신 알림창 피드백을 제공합니다.

## 검증 결과
- **등록 테스트**: 욕설이 포함된 리뷰 등록 시 "비속어가 포함된 내용은 등록할 수 없습니다." 알림과 함께 차단됨을 확인하였습니다.
- **수정 테스트**: 기존 정상 리뷰를 욕설로 수정 시도 시 동일하게 차단됨을 확인하였습니다.
- **AI 분석**: 욕설이 없는 교묘한 비방글은 정상 저장된 후 AI가 위험 점수를 매겨 관리자 대시보드에 정상 보고됨을 확인하였습니다.
