# Report: 예약 취소 페이지 UI 레이아웃 통일

- **Date**: 2026-03-24
- **Reporter**: parkcoding
- **Task**: 마이페이지 예약 취소 신청 페이지 디자인을 변경 신청 페이지 스타일로 개편

## 📝 작업 요약
예약 취소 신청 페이지(`reservation-cancel.mustache`)의 디자인이 예약 변경 신청 페이지(`reservation-change.mustache`)와 상이하여 사용자 경험의 일관성이 떨어졌습니다. 이를 해결하기 위해 `reservation-change`의 카드 기반 레이아웃과 스타일을 이식했습니다.

## 🛠️ 변경 사항
1. **Layout Structure**:
   - 기존의 단일 카드 구조에서 '상단 기존 정보 섹션'과 '하단 취소 요청 폼 섹션'의 2단 구성으로 변경했습니다.
   - 사이드바 내비게이션을 포함한 마이페이지 표준 레이아웃을 준수했습니다.
2. **Components**:
   - 상단 섹션에 예약된 사이트 이미지를 노출하여 직관성을 높였습니다.
   - 환불 계좌 정보 입력란을 `bg-light` 배경의 둥근 모서리 박스로 그룹화하여 깔끔하게 정리했습니다.
   - '취소 전 유의사항' 배너를 변경 페이지의 `info-banner`(Dashed Border) 스타일로 통일하되, 취소의 위험성을 강조하기 위해 `warning`(노란색) 테마를 적용했습니다.
3. **Buttons**:
   - 하단 버튼 그룹을 2분할 가로형으로 배치하고 `danger` 색상을 적용하여 가시성을 확보했습니다.

## ✅ 검증 결과
- `reservation.siteImage`, `id`, `siteName` 등 필수 데이터가 새로운 레이아웃에 정확히 매핑됨을 확인했습니다.
- 예약 변경 페이지와 레이아웃이 1:1로 매칭되어 마이페이지 전반의 UI 완성도가 향상되었습니다.
