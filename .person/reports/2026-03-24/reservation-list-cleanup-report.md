# Report: 예약 목록 UI 간소화 (상세 보기 버튼 제거)

- **Date**: 2026-03-24
- **Reporter**: parkcoding
- **Task**: 마이페이지 예약 목록 내 중복 버튼 제거

## 📝 작업 요약
예약 목록 페이지(`reservations.mustache`)에서 각 예약 카드 클릭 시 상세 페이지로 이동하는 기능(`onclick`)이 이미 구현되어 있어, 중복되는 '상세 보기' 버튼을 제거하여 UI를 간소화했습니다.

## 🛠️ 변경 사항
1. **reservations.mustache**:
   - `{{#canModify}}` 섹션 내의 '상세 보기' 버튼(`<a href="...">상세 보기</a>`)을 삭제했습니다.
   - 기존의 '예약 변경' 및 '예약 취소' 버튼 레이아웃은 그대로 유지했습니다.

## ✅ 검증 결과
- 카드 자체의 `onclick` 이벤트(`location.href='/mypage/reservations/{{id}}/detail'`)를 통해 상세 페이지 진입이 정상적으로 작동함을 확인했습니다.
- 버튼이 제거됨으로써 예약 변경/취소 액션에 더 집중할 수 있는 깔끔한 UI가 되었습니다.
