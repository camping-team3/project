## 작업 요약
- 관리자 페이지 내 '구역/사이트 통합 관리' 및 '시즌별 요금 정책' 화면 접속 불가 문제 해결.
- 사이드바 메뉴 누락 수정 및 개발용 자동 로그인 계정 권한 조정.

## 변경 사항
1. **사이드바 메뉴 추가 (`admin-header.mustache`)**:
   - 기존에 누락되었던 '구역/사이트 관리'(`/admin/sites`) 링크를 사이드바에 추가하여 접근성을 확보했습니다.
2. **자동 로그인 계정 변경 (`application-dev.properties`)**:
   - 개발 편의를 위한 `dev.auto-login.username`을 기존 `ssar`(일반 유저)에서 `admin`(관리자)으로 변경했습니다. 
   - 이를 통해 관리자 페이지 접근 시 권한 부족으로 인한 차단을 방지했습니다.

## 검증 결과
- `/admin/sites` (구역/사이트 관리)와 `/admin/sites/season` (시즌 관리) 경로가 정상적으로 매핑되어 있으며, 사이드바를 통해 편리하게 이동할 수 있음을 확인했습니다.
- 자동 로그인 시 `sessionUser`가 `ADMIN` 권한을 가진 관리자 계정으로 설정되어 모든 관리자 기능을 즉시 테스트할 수 있습니다.
