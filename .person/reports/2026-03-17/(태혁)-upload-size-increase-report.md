# [설정] 멀티파트 파일 업로드 용량 제한 상향 보고서

## 1. 개요
공지사항 등 서비스 내에서 고화질 이미지 업로드 시 발생하는 용량 제한 오류를 해결하기 위해 Spring Boot의 파일 업로드 설정 값을 조정하였습니다.

## 2. 주요 변경 사항
- **application.properties**:
    - `spring.servlet.multipart.max-file-size`: 1MB(기본값)에서 **10MB**로 상향.
    - `spring.servlet.multipart.max-request-size`: 10MB(기본값)에서 **100MB**로 상향.

## 3. 검증 결과
- 공통 설정(`application.properties`)에 적용하여 `dev` 및 `prod` 프로필 모두에 일관된 제한 값이 적용됨을 확인.
- 1MB를 초과하는 중대형 이미지 파일도 서버에서 정상적으로 수신 및 저장 가능함을 설정 수준에서 보장.

## 4. 비유로 설명하기
이전의 파일 업로드는 마치 **우체통 입구가 너무 좁아서 얇은 편지만 들어가는 상태**였습니다. 요즘처럼 사진이 많이 든 두꺼운 편지(고화질 이미지)를 보내려고 하면 입구에서 걸려버렸죠.

이번 수정을 통해, **우체통 입구를 넉넉하게 넓혔습니다.** 이제는 한 통의 두꺼운 편지는 물론, 여러 통을 한꺼번에 부쳐도 우체국(서버)에서 문제없이 받아낼 수 있게 되었습니다.
